WebCLIPS -- Error Reporting Facility
The error number is : BLOD0001
The error message is : CLIPS failed while loading this binary image file -- Read Failure


Explanation :

While loading a binary Image file for a screen, WebCLIPS failed while attempting to BLoad a file. This error usually indicates that WebCLIPS could not find the file.


Resolution :

Please update the location of the file. It is located in one of two places :

  1. In WebCLIPS.INI under the section for the screen name.
  2. It was a fact asserted from a previous screen.
