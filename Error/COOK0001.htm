WebCLIPS -- Error Reporting Facility
The error number is : COOK0001
The error message is : Could not find a corrsponding SaveFactsFor entry in WebCLIPS.INI for this screen


Explanation :

When facts are saved to disk for future processing, WebCLIPS needs to know which screen the facts are being saved for.


Resolution :

Specify a target screen in WebCLIPS.INI
