WebCLIPS -- Error Reporting Facility
The error number is : COOK0002
The error message is : Could not generate a temporary filename for Cookie processing


Explanation :

In order redirect routed output to the screen, WebCLIPS uses _tempnam() to generate temporary filenames. In this case, this function failed.


Resolution :

The most likely cause of the problem is that the web server does not have proper authority to create files to the Windows Temp Directory. If this is the case, contact your security administrator to resolve the problem.
