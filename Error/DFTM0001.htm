WebCLIPS -- Error Reporting Facility
The error number is : DFTM0001
The error message is : CLIPS failed while loading the deftemplate script -- Read Failure.


Explanation :

In order to properly perform dynamic screen navigation, WebCLIPS needs to have deftemplates in place to process certain facts. This error usually indicates that WebCLIPS could not find the file.


Resolution :

The location of the deftemplates is dictated by the Deftemplates entry in the [System] section of WebCLIPS.INI. Please make sure WebCLIPS.INI is pointing to the proper location.
