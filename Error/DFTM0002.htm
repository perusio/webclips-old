WebCLIPS -- Error Reporting Facility
The error number is : DFTM0002
The error message is : CLIPS failed while loading the deftemplate script -- Bad Load Name.


Explanation :

WebCLIPS tried to load the deftemplates it needs to perform dynamic screen navigation but failed due to a malformed file name. The location of the deftemplates is dictated by the Deftemplates entry in the [System] section of WebCLIPS.INI.


Resolution :

Update the file name in WebCLIPS.INI to a proper format.
