WebCLIPS -- Error Reporting Facility
The error number is : DFTM0004
The error message is : CLIPS failed while loading the deftemplate script -- Parse Failure.


Explanation :

WebCLIPS attempted to load the deftemplates it requires to perform dynamic screen navigation but failed due to a syntax error. The location of the deftemplates is dictated by the Deftemplates entry in the [System] section of WebCLIPS.INI.


Resolution :

Correct the syntax error.
