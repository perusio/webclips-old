WebCLIPS -- Error Reporting Facility
The error number is : DRBL0002
The error message is : WebCLIPS failed to open the file for rerouted output for CLIPS


Explanation :

WebCLIPS captures CLIPS output to be redirected to the web server. In order to do this WebCLIPS stores this data in a temporary file.


Resolution :

WebCLIPS was unable to open the temporary file. There could be several reasons for this :

  1. WebCLIPS may not have authorization to read the temporary file.
  2. The temporary file was deleted/renamed.
  3. The file may be locked.
