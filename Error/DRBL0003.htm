WebCLIPS -- Error Reporting Facility
The error number is : DRBL0003
The error message is : WebCLIPS failed to close the file for rerouted output for CLIPS


Explanation :

WebCLIPS captures CLIPS output to be redirected to the web server. In order to do this WebCLIPS stores this data in a temporary file.


Resolution :

WebCLIPS was unable to close the temporary file. There could be several reasons for this :

  1. The temporary file was deleted/renamed.
  2. The file may be locked.
