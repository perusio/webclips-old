WebCLIPS -- Error Reporting Facility
The error number is : DRBL0007
The error message is : Could not delete this Router file


Explanation :

After CLIPS processing, WebCLIPS deletes any temporary files it creates.


Resolution :

WebCLIPS was unable to delete a temporary file. There could be several reasons for this :

  1. WebCLIPS may not have authorization to read the temporary file.
  2. The temporary file was deleted/renamed.
  3. The file may be locked.
