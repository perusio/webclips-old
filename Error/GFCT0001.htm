WebCLIPS -- Error Reporting Facility
The error number is : GFCT0001
The error message is : CLIPS failed while loading this script -- Read Failure


Explanation :

While loading a group of facts from a text file (using load-facts) WebCLIPS encountered a problem with the text file. This error usually indicates that WebCLIPS could not find the file.


Resolution :

Please update the location of the file. It is located in one of two places :

  1. In WebCLIPS.INI under the section for the screen name.
  2. It was a fact asserted from a previous screen.
