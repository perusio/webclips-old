WebCLIPS -- Error Reporting Facility
The error number is : GFCT0003
The error message is : CLIPS failed while loading this script -- Bad Load Name


Explanation :

WebCLIPS tried to load a text file containing a group of facts (using load-facts) but failed due to a malformed file name.


Resolution :

The location of the file can be in one of two places :

  1. In WebCLIPS.INI under the section for the screen name.
  2. It was asserted as a facts from a previous WebCLIPS screen.
Make sure the name is properly formed.
