WebCLIPS -- Error Reporting Facility
The error number is : HTML0001
The error message is : CLIPS failed while loading the HTML helper script -- Read Failure.


Explanation :

Certain screens utilize the WebCLIPS HTML Helper functions to dynamically generate screens. This error usually indicates that WebCLIPS could not find the file that contains the HTML Helper functions.


Resolution :

The location of the HTML Helper functions is dictated by the HelperFileName entry in the [System] section of WebCLIPS.INI. Please make sure WebCLIPS.INI is pointing to the proper location.
