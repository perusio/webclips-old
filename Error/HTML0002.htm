WebCLIPS -- Error Reporting Facility
The error number is : HTML0002
The error message is : CLIPS failed while loading the HTML helper script -- Bad Load Name.


Explanation :

WebCLIPS tried to load the HTML Helper functions for a screen but failed due to a malformed file name. The location of the HTML Helper functions is dictated by the HelperFileName entry in the [System] section of WebCLIPS.INI.


Resolution :

Update the file name in WebCLIPS.INI to a proper format.
