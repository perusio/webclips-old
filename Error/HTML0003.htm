WebCLIPS -- Error Reporting Facility
The error number is : HTML0003
The error message is : CLIPS failed while loading the HTML helper script -- CLIPS not initialized.


Explanation :

WebCLIPS requires that the Windows NT/95 DLL be loaded and available to the web server.


Resolution :

WebCLIPS loads the CLIPS Inference Engine automatically. Something external to WebCLIPS must be unloading CLIPS.DLL
