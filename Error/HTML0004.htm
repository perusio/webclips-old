WebCLIPS -- Error Reporting Facility
The error number is : HTML0004
The error message is : CLIPS failed while loading the HTML helper script -- Parse Failure.


Explanation :

WebCLIPS attempted to load the HTML Helper functions for a screen but failed due to a syntax error. The location of the HTML Helper functions is dictated by the HelperFileName entry in the [System] section of WebCLIPS.INI.


Resolution :

Correct the syntax error.
