WebCLIPS -- Error Reporting Facility
The error number is : IFCT0003
The error message is : Fact Assertion from web server data failed -- split up fact name/value pair


Explanation :

WebCLIPS was unable to assert a fact that was split up across two fields in an HTML file.


Resolution :

There are several details areas where fact assertion can fail :

  1. All factnameX tags must have corresponding factvalueX tags. For example a factname42 tag must have a factvalue42 tag.
  2. The fact was improperly formed. This could stem from a '(' or a ')' missing or a data field was specified without data ... etc. etc.
  3. CLIPS has run out of memory.
  4. CLIPS has run out of 'fact identifiers'.
