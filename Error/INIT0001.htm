WebCLIPS -- Error Reporting Facility
The error number is : INIT0001
The error message is : CLIPS failed during initialization


Explanation :

This error usually appears for one of two reasons :

  1. Insufficient Memory
  2. WebCLIPS is unable to find CLIPS.DLL


Resolution :

If there is insufficient memory, then some applications must be unloaded on the web server.

In order for the web server to find CLIPS.DLL, it must reside in one of three places :

  1. The web servers /CGI-BIN/ (or equivalent) directory
  2. The WINDOWS\SYSTEM directory
  3. In a directory specified by the PATH environment variable
