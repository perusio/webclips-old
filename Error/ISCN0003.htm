WebCLIPS -- Error Reporting Facility
The error number is : ISCN0003
The error message is : Could not read this Initial Screen file


Explanation :

WebCLIPS screens are sometimes dictated by WebCLIPS.INI. This screen is in the form of an HTML file. WebCLIPS was unable to read this HTML file.


Resolution :

There could be several reasons for this :

  1. WebCLIPS may not have authorization to read the temporary file.
  2. The temporary file was deleted/renamed.
  3. The file may be locked.
