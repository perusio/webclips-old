WebCLIPS -- Error Reporting Facility
The error number is : NAVG0001
The error message is : Could not assert a new screen name from the INI file


Explanation :

WebCLIPS was unable to assert a fact for a new screen name. The source of the new screen name is a NextScreenName entry in WebCLIPS.INI.


Resolution :

There are three possible reasons why this error would occur :

  1. The fact was improperly formed. This could stem from a '(' or a ')' missing or a data field was specified without data ... etc. etc.
  2. CLIPS has run out of memory.
  3. CLIPS has run out of 'fact identifiers'.
