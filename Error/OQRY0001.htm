WebCLIPS -- Error Reporting Facility
The error number is : OQRY0001
The error message is : Could not open this ODBC Query file


Explanation :

A screen specified to WebCLIPS that an ODBC query should be run. For this particular case, the query is contained in a file. WebCLIPS could not find the file.


Resolution :

There are two sources for ODBC queries :

  1. In WebCLIPS.INI. Update the query contained in this file appropriately.
  2. In a fact asserted from a previous screen. Find the appropriate CLIPS script and make the necessary changes.
