WebCLIPS -- Error Reporting Facility
The error number is : OQRY0002
The error message is : Could not allocate memory for this ODBC Query file


Explanation :


Resolution :

A screen specified to WebCLIPS that an ODBC query should be run. For this particular case, the query is contained in a file. The web server could not allocate enough memory to read in this file.


Resolution :

Some application(s) must be unloaded on the web server.
