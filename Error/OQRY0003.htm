WebCLIPS -- Error Reporting Facility
The error number is : OQRY0003
The error message is : Could not read this ODBC Query file


Explanation :

A screen specified to WebCLIPS that an ODBC query should be run. For this particular case, the query is contained in a file. WebCLIPS experienced an I/O problem while reading the file.


Resolution :

This problem usually occurs in one of two ways :

  1. The web server does not have proper authority to read the file. If this is the case, contact your security administrator to resolve the problem.
  2. A low level (i.e. operating system) problem occurred. Contact your system administrator to help resolve the problem.
