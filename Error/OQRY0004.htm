WebCLIPS -- Error Reporting Facility
The error number is : OQRY0004
The error message is : Could not find either file or query parameter in this ODBC request


Explanation :

A screen specified to WebCLIPS that an ODBC query should be run. For this particular case, the screen specified neither a QUERY or a FILE containing a query to WebCLIPS.


Resolution :

If you are specifying an ODBC query with the SQL embedded in the deftemplate, then you must place your SQL in the 'query' slot. For example,

(odbcquery (query SELECT * FROM tablename)
           (connect ODBC-Connect-Info;)
           (deftemplatename table-name)
           (implodedata Y))
If you are specifying an ODBC query with the SQL contained in a separate file, then you must place the file name in the 'file' slot. For example,
(odbcqueryfile (file c:\websrvr\sql\query1.txt)
               (connect ODBC-Connect-Info;)
               (deftemplatename table-name)
               (implodedata N))
