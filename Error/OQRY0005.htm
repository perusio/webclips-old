WebCLIPS -- Error Reporting Facility
The error number is : OQRY0005
The error message is : Could not find the connect parameter in this ODBC request


Explanation :

A screen specified to WebCLIPS that an ODBC query should be run. For this particular case, the screen did not provide ODBC CONNECT information to WebCLIPS.


Resolution :

Please provide the necessary ODBC connect information for the query.
