WebCLIPS -- Error Reporting Facility
The error number is : OQRY0006
The error message is : Could not find the deftemplatename parameter in this ODBC request


Explanation :

A screen specified to WebCLIPS that an ODBC query should be run. For this particular case, the screen did not provide a deftemplate name to WebCLIPS.


Resolution :

Please specify the deftemplate name.
