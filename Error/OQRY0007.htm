WebCLIPS -- Error Reporting Facility
The error number is : OQRY0007
The error message is : Could not find the Implode data parameter in this ODBC request


Explanation :

A screen specified to WebCLIPS that an ODBC query should be run. For this particular case, the screen did not provide the IMPLODE DATA parameter to WebCLIPS.


Resolution :

Please specify the IMPLODE DATA parameter (Y/N).
