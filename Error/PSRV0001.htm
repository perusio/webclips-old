WebCLIPS -- Error Reporting Facility
The error number is : PSRV0001
The error message is : WebCLIPS failed to generate a unique file name for a temporary fact file


Explanation :

WebCLIPS captures CLIPS output to be saved between screens. In order to do this WebCLIPS stores this data in a temporary file and must therefore generate a temp file name.


Resolution :

The most likely cause of the problem is that the web server does not have proper authority to create files to the Windows Temp Directory. If this is the case, contact your security administrator to resolve the problem.
