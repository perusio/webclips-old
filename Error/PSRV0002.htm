WebCLIPS -- Error Reporting Facility
The error number is : PSRV0002
The error message is : Could not open this temporary fact file


Explanation :

WebCLIPS captures CLIPS output to be saved between screens. In order to do this WebCLIPS stores this data in a temporary file.


Resolution :

After CLIPS processing, WebCLIPS was unable to open the temporary file. There could be several reasons for this :

  1. WebCLIPS may not have authorization to read the temporary file.
  2. The temporary file was deleted/renamed.
  3. The file may be locked.
