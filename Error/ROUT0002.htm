WebCLIPS -- Error Reporting Facility
The error number is : ROUT0002
The error message is : WebCLIPS failed during the setup for this System Router


Explanation :

An attempt to redirect the output of a router to a file failed.


Resolution :

This is most a file related error. Either the disk is full, the web server does not have proper authority to use the file, malfomed file name etc. etc.
