WebCLIPS -- Error Reporting Facility
The error number is : ROUT0003
The error message is : WebCLIPS failed during the setup for this User-defined Router


Explanation :

An attempt to open a user-defined router via (open FileName RouterName "w") failed.


Resolution :

Possible causes of this problem include :

  1. Malformed File Name
  2. Malformed Router Name
  3. Web server file authority or disk space
