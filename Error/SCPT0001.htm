WebCLIPS -- Error Reporting Facility
The error number is : SCPT0001
The error message is : WebCLIPS requires either a script or a binary image file to execute.


Explanation :

In order to execute properly WebCLIPS needs either a binary image file that contains rules or a normal text file that contains rules.


Resolution :

Please supply one of the two required file types.

Under the section for the screen name, make sure the first script name in WebCLIPS.INI is pointed to by the entry script1.
