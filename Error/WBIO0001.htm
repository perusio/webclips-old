WebCLIPS -- Error Reporting Facility
The error number is : WBIO0001
The error message is : Invalid request method


Explanation :

WebCLIPS was executed in a manner it didn't expect. This most likely means that someone attempted to run WebCLIPS from the command line in DOS.


Resolution :

WebCLIPS can be invoked using either the GET or POST method. Please execute WebCLIPS using one of these two methods.
