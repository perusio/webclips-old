WebCLIPS -- Error Reporting Facility
The error number is : WBIO0003
The error message is : No Input Screen Specified from method:GET


Explanation :

WebCLIPS needs to know which screen to execute. The screen name was not present in the data from the web server and the input method was POST.


Resolution :

Recode your HTML document or your CLIPS script to ensure that a valid WebCLIPS screen name is present. These valid screen names are listed in WebCLIPS.INI
