CLIPS (C Language Integrated Production System) is an expert systems language that allows you to define rules and facts in a script. The script is then executed through an 'inference' engine to arrive at a decision(s). For more information, please visit the CLIPS Home Page.

WebCLIPS is an implementation of CLIPS run as a CGI application under Windows NT/95. By utilizing the 32 bit Windows NT/95 DLL , standard CLIPS functions were made available via MFC C++ wrapper classes. The WebCLIPS scripts are standard to the CLIPS specification. The only 'unusual' element is that the results generated from WebCLIPS screens have some embedded HTML tags to properly display the output. A fuller explanation of WebCLIPS features are available as well as download and setup information.
